1. System Requirements

PHP Requirements

  • PHP 8.2 or higher
  • Memory Limit: 512MB+
  • Max Execution Time: 300s+

PHP Extensions

  • OpenSSL
  • PDO MySQL
  • MBString
  • Tokenizer
  • XML
  • Ctype
  • JSON
  • Fileinfo
  • GD Library
  • ZIP

Database

  • MySQL 5.7+ or MariaDB 10.3+
  • Remote database support
  • Dual database option

Server

  • Apache 2.4+ or Nginx
  • mod_rewrite enabled
  • SSL/TLS recommended
  • Node.js (optional)

Checking Requirements

The installer will automatically check all requirements before proceeding. If any requirements are missing, you'll see a detailed error message with instructions.

üîí

Ensure mod_rewrite is enabled and .htaccess is properly configured

Permission Fixes

# Linux/Unix Permissions chmod -R 775 storage bootstrap/cache chown -R www-data:www-data storage bootstrap/cache # If using Laravel Sail or Docker chmod -R 775 storage bootstrap/cache chmod -R 777 storage/framework/

Manual .env Setup (If Needed)

If the automatic .env creation fails, create the file manually:

APP_NAME="JPOS System" APP_ENV=production APP_KEY= APP_DEBUG=false APP_URL=https://your-domain.com DB_CONNECTION=mysql DB_HOST=localhost DB_PORT=3306 DB_DATABASE=jpos_system DB_USERNAME=root DB_PASSWORD= # Run this command after creating .env php artisan key:generate

5. Security Recommendations

üîí Security Checklist

  • Change default admin password immediately
  • Enable HTTPS/SSL for your domain
  • Regularly update the application
  • Keep server software updated
  • Configure proper file permissions
  • Set up regular backups
  • Monitor access logs

üìù Important Notes

  • The installer deletes itself after successful installation
  • Do not restore or reuse the installer file
  • Keep the storage/installed.lock file intact
  • Never expose your .env file publicly
  • Regular backups are recommended
