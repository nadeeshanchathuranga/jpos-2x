Getting Started with jPOS

Learn how to access the system and navigate the dashboard.

Login & Authentication

To access the jPOS system, you need valid credentials provided by your system administrator.

Step 1: Navigate to Login Page

Open your web browser and navigate to the jPOS URL provided by your administrator.

Step 2: Enter Login Credentials

Use one of the following default accounts to log in:

Admin User

  • Email: admin@gmail.com
  • Password: 123456789

Manager User

  • Email: manager@gmail.com
  • Password: 123456789

Cashier User

  • Email: cashier@gmail.com
  • Password: 123456789

Store Keeper User

  • Email: stockkeeper@gmail.com
  • Password: 123456789

Then:

  • Enter the email address
  • Enter the password
  • Click the "Sign In" button
Note: If you forget your password, contact your system administrator for a password reset.

Dashboard Overview

After successful login, you'll be redirected to the main dashboard which provides an overview of your business operations.

Dashboard Components:

The dashboard shows role-based tiles for quick access to core modules: Inventory (Products, Brands, Categories, Units, Types), Purchasing & Stock (POR, GRN, Returns, Transfers), Sales/POS, and rich Reports.

Admin-only areas include System Users and Settings (Company Info, App, SMTP), with each card linking directly to its respective feature.
