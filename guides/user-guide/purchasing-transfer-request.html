Product Transfer Request

Request inventory transfers between locations or warehouses.

Create a Product Transfer Request

  1. Go to Purchasing & Stock ‚Üí Product Transfer Request
  2. Click "Create Transfer Request"
  3. Enter request details:
    • From Location: Source location
    • To Location: Destination location
    • Transfer Date: Date for transfer
    • Remarks: Optional notes
  4. Add products and quantities to transfer
  5. Save the transfer request
