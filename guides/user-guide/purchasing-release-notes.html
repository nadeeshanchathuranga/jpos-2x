Product Release Notes

Record product releases from storage/production into active inventory.

Create Product Release Note

  1. Go to Purchasing & Stock ‚Üí Product Release Notes
  2. Click "Create Release Note"
  3. Enter release details:
    • Release Date: Date of release
    • From Location: Storage/production location
    • To Location: Active inventory location
    • Remarks: Optional notes
  4. Add products and quantities released
  5. Save to update stock
