Stock Return

Return stock to suppliers due to defects, damage, or other issues.

Create Stock Return

  1. Go to Purchasing & Stock ‚Üí Stock Returns
  2. Click "Create Stock Return"
  3. Enter return details:
    • Supplier: Supplier to return to
    • Return Date: Date of return
    • Reference GRN: Link to the original GRN (recommended)
    • Remarks: Optional notes
  4. Add products and quantities to return
  5. Save to update stock and create credit note

  • Always reference the original GRN for accuracy
  • Document reasons for returns for auditing
