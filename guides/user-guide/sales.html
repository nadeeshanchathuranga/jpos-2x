Sales Management

Manage all your sales transactions, invoices, and customer relationships.

Customers

Fast checkout system with barcode scanning and multiple payment options.

Adding a Customer

  1. Go to Sales ‚Üí Customers
  2. Click "Add Customer"
  3. Enter customer details (name, phone, email, address)
  4. Save the customer

Discount

View, edit, and manage all discount from this section.

Discounts

  1. Go to Sales ‚Üí Discount
  2. View discount list with:
    • Discount Name: Name or code of the discount
    • Type: Percentage, Fixed Amount, or Buy X Get Y
    • Value: Discount value (e.g., 10% or Rs. 500)
    • Total: Total discount amount applied
    • Start Date: When the discount becomes active
    • End Date: When the discount expires
    • Status: Active, Scheduled, or Expired

Tax Management

Adding a Tax

  1. Go to Sales ‚Üí Taxes

    • Tax Name: Tax name (required)

Creating a New Sale / Bill

Process sales transactions with barcode scanning, product selection, and multiple payment options.

Step-by-Step Sale Process

  1. Navigate to Sales ‚Üí New Sale or Point of Sale
  2. You will see the New Sale/Bill screen with:
    • Scan Barcode: Input field for quick product entry
    • Customer & Date: Select customer and date
    • Price Type: Choose between Retail or Wholesale pricing
    • Add Products: Browse and select products
  3. Add products to cart by:
    • Scanning barcode directly, or
    • Clicking "Browse Products" button to search and select
  4. View cart items showing:
    • Product name and details
    • Unit price
    • Quantity (adjustable)
    • Total for each item
    • Remove/Edit actions
  5. Review Bill Summary on the right showing:
    • Total Amount: Sum of all items
    • Discount: Apply discount if applicable
    • Net Amount: Final amount after discount
    • Change: Amount returned to customer
  6. Click "Add Payment" to process payment with options:
    • Cash
    • Card
    • Check
    • Bank Transfer
    • Multiple payment methods
  7. Click "Complete Sale (F9)" to finalize the transaction
  8. Invoice will be generated and can be printed

Price Types

  • Retail: Standard retail pricing for individual customers
  • Wholesale: Discounted pricing for bulk purchases and wholesale customers

Keyboard Shortcuts

  • F9: Complete Sale
  • F8: Clear Cart
  • ESC: Focus Barcode Scanner

Product Returns

Process returns for previously sold products. Find the original sale and create a return transaction to reverse the sale and restore inventory.

Creating a Product Return

  1. Navigate to Sales ‚Üí Product Returns
  2. Click "Create Return" button
  3. You will see the "Create Product Return" screen with search section:
    • Search: Find by Customer name, Sale No, or Product name
    • From Date: Set start date for search range
    • To Date: Set end date for search range
  4. Click "Search" button to find available sales products
  5. Review the "Available Sales Products" table showing:
    • Product: Product name and SKU
    • Sale No: Original invoice number and date
    • Customer: Customer who purchased the item
    • Qty Sold: Quantity that was sold
    • Price: Original selling price
    • Action: "+ Add" button to add to return
  6. Click "+ Add" for each product you want to return
  7. Specify the quantity to return for each product
  8. Review return details
  9. Click "Create Return" to process the return

Return Process Details

  • Search Functionality: Filter sales by customer name, invoice number, or specific product
  • Date Range: Narrow search by specifying From and To dates
  • Multiple Returns: You can add multiple products to a single return transaction
  • Partial Returns: Return part or all of the quantity that was sold
  • Automatic Stock Update: Stock is automatically restored when return is created

Return Effects

When you create a product return:

  • Stock inventory is increased by the returned quantity
  • Sales revenue is reversed
  • Customer credit/refund is processed
  • Original invoice reference is maintained
  • Return transaction is recorded for audit trail
